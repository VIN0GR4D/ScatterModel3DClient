Параметр freqBand отвечает за выбор диапазона частот для радара. Он определяет диапазон частот, на которых работает радар, а также шаги дискретизации по осям X, Y и Z. Эти шаги дискретизации используются при расчетах, связанных с волновыми характеристиками.

Диапазон значений freqBand
Параметр freqBand может принимать следующие значения:

0 - P-диапазон (400-450 МГц)
1 - L-диапазон (1-1.5 ГГц)
2 - S-диапазон (2.75-3.15 ГГц)
3 - C-диапазон (5-5.5 ГГц)
4 - X-диапазон (9-10 ГГц)
5 - Ka-диапазон (36.5-38.5 ГГц)

Обработка на сервере
На сервере freqBand используется для настройки волновых характеристик радара. Вот как это происходит:

1. Создание объекта radar_wave:
radar_wave wave1(freqband);

2. Настройка частотного диапазона и шагов дискретизации:
Внутри конструктора radar_wave в зависимости от значения freqBand устанавливаются значения частотного диапазона (fmin и fmax) и шаги дискретизации по осям X и Y.

3. Вычисление длины волны:
Средняя частота диапазона используется для вычисления длины волны:
    m_stepZ = m_stepX; //азимут. и угломестн. разрешение одинаковы
    f0 = (fmin + fmax)/2;
    lambda = c/f0;     //m

4. Настройка шагов дискретизации:
Шаги дискретизации устанавливаются на основе диапазона частот:
m_stepX = ...;
m_stepY = ...;
m_stepZ = m_stepX; //азимут. и угломестн. разрешение одинаковы
